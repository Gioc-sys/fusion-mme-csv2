# Fusion MME CSV

Application pour fusionner des fichiers MME et ajouter des voies CSV.

## 🚀 Télécharger l'EXE

1. Aller dans **Actions** (en haut)
2. Cliquer sur le dernier workflow vert ✅
3. Descendre jusqu'à **Artifacts**
4. Télécharger **FusionMMECSV-Windows**
5. Extraire le ZIP
6. Double-cliquer sur `FusionMMECSV.exe`

## 📋 Fonctionnalités

- **Ajout de Voie MME/CSV** : Ajouter des voies depuis un fichier CSV
- **Fusion de MME** : Fusionner plusieurs fichiers MME
- Conservation de l'arborescence
- Interface web automatique
